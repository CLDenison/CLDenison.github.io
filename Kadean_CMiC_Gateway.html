Overview

The Project Gateway enables CMiC Field clients to easily invite their project team to collaborate on any project to which they have been invited, using their Single Sign-On (SSO) credentials.

When a collaborator signs in to CMiC Collaborate, the Project Gateway Dashboard is presented. If a collaborator has been invited to work on multiple projects by multiple CMiC Field users at Kadean, the dashboard lists the projects to which the collaborator has been invited. Selecting a project from the dashboard navigates the user to the CMiC Field portal.

Publish Projects to Project Gateway Dashboard

To publish a project to the Project Gateway system:

  • Launch the Project Maintenance screen by clicking on the project name at the top of the left menu (above Project Setup).
  • Select the Gateway tab.
  • Click Edit Publishing on the toolbar.
  • Select the Private radio button.
  • Project Expires After:
    • If Yes is selected, enter the number of days before expiration.
    • If No is selected, the project will not automatically expire.
    • No is typically selected.
  • Default Project Role: Subcontractor (recommended).
  • Default Security Role: Subcontractor (recommended).
  • Default LDAP Server: Select Default.
  • Send All Project Contacts: Select No. Never select Yes.
  • Gateway SSO Passthrough: Should be checked.
  • Invitation Template:
    • Copy the contents of this file by opening the file, selecting all of the text, and closing the file
  • Click Publish to Collaborate.

Send Invitations to Project Gateway

After a project is published, collaborators (Subcontractors, Owners, Architects, and Engineers) can be invited from the Gateway tab.

  1. Click Select in the To Be Invited section.
  2. Choose Project Contacts in the pop-up window.
  3. Select the collaborators to invite.
  4. Click Accept.
  5. Click Send All or select individuals and click Send.
  6. Select the Project invitation template.
  7. Click Send.
  8. Confirm the success message.
  9. Invited collaborators appear in the Invited Contacts section.
